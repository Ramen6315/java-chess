# java-chess
체스 게임 구현을 위한 저장소


#기능 목록
- [x] 체스판 초기화
    - [x] 팀에 따른 체스말 표기
    - [x] 체스말이 없는 부분 표기 
    - [x] 포지션에 따른 체스말 배치
- [x] 체스 시작 종료를 입력받는다. 
- [x] 체스판 출력 
- [ ] 체스말 이동 여부 판단
    - [ ] 룩
    - [ ] 비숍
    - [ ] 퀸
    - [ ] 킹
    - [ ] 나이트
    - [ ] 폰

#예외 처리
- [x] 시작, 종료값 이외의 값을 입력하면 예외처리 한다.
